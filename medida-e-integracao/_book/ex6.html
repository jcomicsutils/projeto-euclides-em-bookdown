Medida e Integração

Exercícios

  1. Demonstrar a Prop. 6.1.1

  2. Demonstrar a Prop. 6.1.3

  3. Provar que a classe dos conjuntos negativos é um \(\sigma\)-anel.

  4. Se \(\mu\) é uma medida com sinal, e \(\nu = \nu^+-\nu^-\), onde \(\nu^+\) e \(\nu^-\) são medidas positivas finitas, então \(\mu^+ \leq \nu^+\) e \(\mu^- \leq \nu^-\).

  5. Se \(f\) é integrável sobre \((\Omega, \mathscr{A}, \mu)\), e \(\nu(A) = \int_A f d\mu\), então

    \[\nu^+(A) = \int_A f^+ d\mu;\] \[\nu^-(A) = \int_A f^- d\mu;\] \[|\nu|(A) = \int_A |f| d\mu.\]

  6. Provar que \(\mu^+(A) = \sup\{\mu(C): C \in \mathscr{A}, C \subseteq A\}\) e \(\mu^-(A) = -\inf\{\mu(C): C \in \mathscr{A}, C \subseteq A\}\).

  7. Seja \(\mathscr{M}^b(\Omega, \mathscr{A}) = \{\mu: \mu \text{ é medida com sinal finita}\}\). Com as operações definidas naturalmente, \(\mathscr{M}^b(\Omega, \mathscr{A})\) é um espaço vetorial. Definimos para cada \(\mu \in \mathscr{M}^b(\Omega, \mathscr{A})\), \(\left\|\mu\right\| = |\mu|(\Omega)\). Provar que com essa norma, \(\mathscr{M}^b(\Omega, \mathscr{A})\) é um espaço de Banach.

  8. Provar que, se \(\mu\) é uma medida com sinal finita, então \[\left\|\mu\right\| = \sup\left\{\left|\int_\Omega f d\mu\right|: |f| \leq 1\right\}.\]

  9. Se \(\nu_1, \nu_2\) e \(\mu\) são medidas com sinal finitas, \(\nu_1 \perp \mu\) e \(\nu_2 \perp \mu\), então \((\nu_1 + \nu_2) \perp \mu\).

  10. Provar que os conjuntos positivos são um \(\sigma\)-anel.

  11. Um conjunto \(N\) é nulo para uma medida com sinal \(\mu\) se, e somente se \(|\mu|(N) = 0\).

  12. Sejam \(\mu\) e \(\nu\) são medidas positivas finitas, e \(\lambda\) uma medida positiva com \(\mu \ll \lambda\) e \(\nu \ll \lambda\). Se \(f = \frac{d\mu}{d\lambda}\) e \(g = \frac{d\nu}{d\lambda}\), resulta que \(\left\|\mu-\nu\right\| = \int_\Omega |f-g| d\lambda\).

  13. Se \(\mu\) e \(\nu\) são duas probabilidades sobre \((\Omega, \mathscr{A})\), então \(\left\|\mu-\nu\right\| = 2 \sup_{A \in \mathscr{A}} |\mu(A)-\nu(A)|\).

  14. Se \(\nu_1\) e \(\nu_2\) são medidas com sinal \(\sigma\)-finitas, \(\mu\) uma medida \(\sigma\)-finita \(\nu_1 \ll \mu\) e \(\nu_2 \ll \mu\), então \(\nu_1 + \nu_2 \ll \mu\), e \[\frac{d(\nu_1 + \nu_2)}{d\mu} = \frac{d\nu_1}{d\mu} + \frac{d\nu_2}{d\mu} \text{ q. t. p. } [\mu].\]

  15. Sejam \(\mu\) e \(\lambda\) duas medidas \(\sigma\)-finitas e \(\nu\) uma medida com sinal \(\sigma\)-finita. Se \(\nu \ll \mu \ll \lambda\), então \[\frac{d\nu}{d\lambda} = \frac{d\nu}{d\mu} \frac{d\mu}{d\lambda} \text{ q. t. p. } [\lambda].\]

  16. Sejam \(\mu\) e \(\lambda\) duas medidas \(\sigma\)-finitas, tais que \(\mu \ll \lambda\). Seja \(f\) uma função quase integrável \(\mu\). Então \[\int_\Omega f d\mu = \int_\Omega f \frac{d\mu}{d\lambda} d\lambda.\]

  17. Se \(\mu\) e \(\nu\) são medidas, tais que \(\mu \equiv \nu\) (isto é, \(\mu \ll \nu \text{ e } \nu \ll \mu\)), então \[\frac{d\mu}{d\nu} = \left(\frac{d\nu}{d\mu}\right)^{-1}.\]

  18. Seja \(\nu \ll \mu\), onde \(\nu\) é uma medida com sinal e \(\mu\) é uma medida \(\sigma\)-finita. Então \[\nu\left(\left[\frac{d\nu}{d\mu} = 0\right]\right) = 0.\]

  19. Seja \(\Omega\) um conjunto não enumerável e \(\mathscr{A} = \{A: A \subseteq \Omega \text{ ou } A^c \text{ é enumerável}\}\). Seja \[\nu(A) = \begin{cases} 1 & A^c \text{ é enumerável} \\ 0 & A \text{ é enumerável} \end{cases}\] Seja \(\mu\) a medida da contagem. Então \(\nu \ll \mu\), mas o Teorema de Radon-Nikodym não é válido.

  20. Seja \(\Omega = [0, 1]\), \(\mathscr{A}\) a \(\sigma\)-álgebra de Borel de \([0, 1]\), \(\nu\) a medida de Lebesgue, \(\mu\) a medida da contagem. Então \(\nu \ll \mu\), mas o Teorema de Radon-Nikodym não é válido.

  21. Seja \(\lambda\) a medida de Lebesgue em \(\mathbb{R}^1\). Seja \[f(x) = \frac{1}{e^{|x|}\sqrt{|x|}}, \text{ para } x \neq 0, \text{ e } +\infty, \text{ para } x = 0.\] Provar

    1. \(\int_{\mathbb{R}^1} f d\lambda < \infty\);

    2. \(\forall \varepsilon > 0 \int_{-\varepsilon}^\varepsilon f^2 d\lambda = +\infty\).

    Seja \(x_1, x_2, \ldots\) uma enumeração dos racionais. Definimos \[h(x) = \sum_{n=1}^\infty \frac{1}{2^n}f(x-x_n);\]

    1. \(\int_{\mathbb{R}^1} h d\lambda < \infty\) (portanto, \(h\) é finita q. t. p. \([\lambda]\))

    Para cada subconjunto \(A\) mensurável Lebesgue, definimos \[\nu(A) = \int_A h^2 d\lambda;\]

    1. \(\nu \ll \lambda\);

    2. \(\nu\) é \(\sigma\)-finita;

    3. Se \(C\) é um intervalo com interior não-vazio \(\nu(C) = +\infty\).

  22. DEFINIÇÃO. Uma medida \(\nu\) definida sobre a \(\sigma\)-álgebra de Lebesgue de \(\mathbb{R}^1\) é dita puramente atômica se existe um conjunto enumerável \(A\), tal que \(\nu(A^c) = 0\). Provar que, se \(\nu\) é uma medida finita sobre a \(\sigma\)-álgebra de Lebesgue de \(\mathbb{R}^1\), \(\nu\) pode ser decomposta de maneira única como soma de três medidas \(\nu_1, \nu_2\) e \(\nu_3\), tal que \(\nu_1 \ll \lambda\), \(\nu_2\) é puramente atômica, e \(\nu_3 \perp \lambda\), com \(\nu_3(\{x\}) = 0, \forall x \in \mathbb{R}^1\).

  23. Sejam \(\mu\) e \(\nu\) duas medidas finitas com sinal. Provar que existe uma medida com sinal que é menor que \(\mu\) e \(\nu\), porém maior que qualquer outra medida com sinal menor que \(\mu\) e \(\nu\). Essa medida é chamada ínfimo de \(\mu\) e \(\nu\) e é denotada com \(\mu \wedge \nu\). Provar também a existência de um supremo \(\mu \vee \nu\). Verifique que \((\mu \vee \nu) + (\mu \wedge \nu) = \mu + \nu\). (Sugestão. \(\mu \wedge \nu = \frac{1}{2}(\mu + \nu - |\mu - \nu|)\)).

  24. Provar que, se \(\mu\) e \(\nu\) são duas medidas positivas, então \(\mu \perp \nu\) se, e somente se, \(\mu \wedge \nu = 0\).

  25. Dar um exemplo de um espaço mensurável \((\Omega, \mathscr{A})\), uma medida finita com sinal \(\nu\) sobre \(\mathscr{A}\), e uma sequência de conjuntos \(\{A_n\}_{n=1,2,\ldots}\) \(A_n \in \mathscr{A} \forall n\), tais que

    \[\nu(\lim \inf A_n) > \lim \inf \nu(A_n);\]

    \[\nu(\lim \sup A_n) < \lim \sup \nu(A_n).\]

    (Sugestão. Seja \((\Omega, \mathscr{A})\) o intervalo \([0, 2]\) com os borelianos de \([0, 2]\). Seja \(\lambda\) a medida de Lebesgue e \(f\) a função

    \[f = I_{[0, 1]} - I_{(1, 2]}.\]

    Seja \(\nu(A) = \int_A f d\lambda\). Seja

    \[A_n = \begin{cases} [0, 2] & n \text{ ímpar.} \\ [0, 1] & n \text{ par.} \end{cases}\]

    É fácil verificar que

    \(\nu(\lim \inf A_n) = \nu([0, 1]) = 1\);

    \(\lim \inf \nu(A_n) = 0\);

    \(\nu(\lim \sup A_n) = \nu([0, 2]) = 0\);

    \(\lim \sup \nu(A_n) = 1\).

  26. Se \(\mu\) é uma medida com sinal finita sobre \((\Omega, \mathscr{A})\) e \(A_n \to A (A_n \in \mathscr{A}, A \in \mathscr{A})\), então \(\mu(A_n) \to \mu(A)\).

  27. Provar que, se \(\mu\) e \(\nu\) são duas medidas finitas sobre \((\Omega, \mathscr{A})\), então \[\sup_{A \in \mathscr{A}} |\mu(A)-\nu(A)| \leq \|\mu-\nu\| \leq 2 \sup_{A \in \mathscr{A}} |\mu(A)-\nu(A)|\] (veja o Exerc. 13).

  28. Se \(\mu\) é uma medida \(\sigma\)-finita sobre \((\Omega, \mathscr{A})\), então existe uma probabilidade \(P\) sobre \((\Omega, \mathscr{A})\), tal que \(\mu \ll P\) e \(P \ll \mu\).

  29. Seja \((\Omega, \mathscr{A})\) um espaço mensurável. Sejam \(\mathscr{F}_1\) e \(\mathscr{F}_2\) duas famílias de medidas finitas sobre \((\Omega, \mathscr{A})\). Dizemos que \(\mathscr{F}_1\) é equivalente a \(\mathscr{F}_2\) (e escrevemos \(\mathscr{F}_1 \equiv \mathscr{F}_2\)), se \(\mu(A) = 0, \forall \mu \in \mathscr{F}_1 \iff \nu(A) = 0, \forall \nu \in \mathscr{F}_2\).

    Provar que, se \(\{\mu_n\}_{n=1,2,\ldots}\) é uma sequência de medidas finitas sobre \((\Omega, \mathscr{A})\), então existe uma probabilidade \(P\) sobre \((\Omega, \mathscr{A})\), tal que \(P \equiv \{\mu_n\}_{n=1,2,\ldots}\).

  30. (Decomposição de Hahn). Definimos \(\mu^-(S) = -\inf\{ \mu(A): A \in \mathscr{A}, A \subseteq S\}\):

    1. Provar que \(\mu^-\) é uma medida sobre \(\mathscr{A}\);

    2. \(\mu^-(\Omega) < \infty\).

    Demonstração. Por i) \(\mu^-\) é em particular aditiva. Então, se \(\mu^-(S) = +\infty\) e \(C \subseteq S\), temos \(\mu^-(C) = +\infty\) ou \(\mu^-(S-C) = +\infty\). Por indução, construímos uma sequência \(\Omega = S_0 \supseteq S_1 \supseteq S_2 \supseteq \ldots\), com \(\mu^-(S_n) = +\infty, \forall n\), escolhendo a cada passo um \(C_n \subseteq S_{n-1}\), tal que \(\mu(C_n) \leq -n\) e definindo \(S_n = C_n\) ou \(S_n = S_{n-1} - C_n\), de maneira que \(\mu^-(S_n) = +\infty\). Como os conjuntos \(S_{n-1} - S_n\) assim obtidos são disjuntos, não podemos ter \(C_n = S_{n-1} - S_n\) para uma coleção infinita de índices \(n\), pois \[-\infty < \mu\left(\sum_{n: C_n=S_{n-1}-S_n} C_n\right) \leq \sum_{n: C_n=S_{n-1}-S_n} (-n).\] Portanto, existe \(N\) tal que \(\forall n \geq N, C_n = S_n\). Então \(\forall n \geq N, \mu(C_n) \leq -n\) \[\mu(\lim S_n) = \lim \mu(S_n) = -\infty,\] o que é uma contradição,

    1. Existe \(B \in \mathscr{A}\), tal que \(\mu^-(B) = \mu^-(\Omega)\).

    Demonstração. \(\forall n\), existe \(S_n\) com \[-\mu^-(S_n) > \mu^-(\Omega) - \frac{1}{2^n}.\] Então \[\mu^-(S_n) > \mu^-(\Omega) - \frac{1}{2^n}\] e portanto, \[\mu^-(\Omega-S_n) < \frac{1}{2^n}.\] Chamando \(T_n = \bigcup\limits_{j=n+1}^\infty (\Omega-S_j)\), temos \[\mu^-(T_n) \leq \sum_{j=n+1}^\infty \frac{1}{2^j} = \frac{1}{2^n}.\] Agora \[\mu^-(S_n-T_n) = \mu^-(S_n)-\mu^-(S_n \cap T_n) \leq \mu^-(S_n) + \mu^-(T_n)\] \[< -\mu^-(\Omega) + \frac{1}{2^n} + \frac{1}{2^n}.\] Como a sequência \(S_n-T_n = \bigcap\limits_{j=n+1}^\infty S_j\) é decrescente, tem-se \[\mu(\lim (S_n-T_n)) = \lim \mu(S_n-T_n) \leq -\mu^-(\Omega).\] Portanto \[\mu^-(\lim (S_n-T_n)) = \mu^-(\Omega).\]

    1. Provar que \((\Omega-B, B)\) é uma decomposição de Hahn para \(\mu\).
