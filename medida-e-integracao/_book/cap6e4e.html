Medida e Integração

6.4 Teorema de Representação de Riesz

Uma importante aplicação do Teorema de Radon-Nikodym é a representação de funcionais lineares limitados sobre \(L_p\); \(1 \leq p < \infty\).

DEFINIÇÃO 6.4.1. Um funcional linear sobre \(L_p, 1 \leq p \leq \infty\) é uma função \(\varphi: L_p \to \mathbb{R}^1\), onde \(\varphi\) é linear.

Se \(\varphi\) é um funcional linear, dizemos que é limitado se existe uma constante \(K\), tal que \[\forall f \in L_p, |\varphi(f)| \leq K \left\|f\right\|_p.\] Definimos também a norma de \(\varphi\) como \[\left\|\varphi\right\| = \sup \left\{|\varphi(f)|: \left\|f\right\|_p \leq 1\right\}.\] Temos o seguinte:

TEOREMA 6.4.1 (Representação de Riesz). Seja \((\Omega, \mathscr{A}, \mu)\) um espaço de medida \(\sigma\)-finita. Seja \(1 \leq p \leq \infty\) e \(q\) dado pela equação \(\frac{1}{p} + \frac{1}{q} = 1\). Se \(g \in L_q\), então a fórmula \[\varphi(f) = \int_\Omega fg d\mu, f \in L_p,\]

define um funcional linear limitado sobre \(L_p\). Temos também que \(\left\|\varphi\right\| = \left\|g\right\|_q\). Reciprocamente se \(\varphi\) é um funcional linear limitado sobre \(L_p\), \(1 \leq p < \infty\), existe \(g \in L_q\), tal que \(\forall f \in L_p\), \[\varphi(f) = \int_\Omega fg d\mu.\]

Demonstração. Decorre da desigualdade de Hölder que se \(f \in L_p\) e \(g \in L_q\), \(fg \in L_1\) e \[\left|\int_\Omega fg d\mu\right| \leq \left\|f\right\|_p \left\|g\right\|_q.\] Claramente a fórmula \[\varphi(f) = \int_\Omega fg d\mu\] define um funcional linear contínuo com \(\left\|\varphi\right\| \leq \left\|g\right\|_q\). Para provar \(\left\|\varphi\right\| = \left\|g\right\|_q\), consideremos primeiro o caso \(1 < p < \infty\). Seja \[\varepsilon(g) = I_{[g \geq 0]} - I_{[g < 0]}\] e consideramos a função \(\varepsilon(g)|g|^{q-1}\). Temos \[\left\|\varepsilon(g)|g|^{q-1}\right\|_p = \left(\int_\Omega \left(|\varepsilon(g)||g|^{q-1}\right)^p\right)^{1/p} = \left(\int_\Omega |g|^{q}\right)^{1/p} =\] \[= \left\|g\right\|_q^{q/p} = \left\|g\right\|_q^{q-1}.\] Por outra parte, \[\left\|\varphi\right\| \left\|\varepsilon(g)|g|^{q-1}\right\|_p \geq \varphi\left(\varepsilon(g)|g|^{q-1}\right) =\] \[= \int_\Omega \varepsilon(g)|g|^{q-1} g d\mu = \int_\Omega |g|^q d\mu =\] \[= \left\|g\right\|_q^q = \left\|g\right\|_q \left\|g\right\|_q^{q-1} =\] \[= \left\|g\right\|_q \left\|\varepsilon(g)|g|^{q-1}\right\|_p,\] o que prova \(\left\|\varphi\right\| \geq \left\|g\right\|_q\).

No caso \(p = \infty\) (\(q = 1\)), temos \(\varepsilon(g) \in L_p\). Resulta \[\varphi(\varepsilon(g)) = \int_\Omega \varepsilon(g) g d\mu =\] \[= \int_\Omega |g| d\mu = \left\|g\right\|_1\] e portanto \(\left\|\varphi\right\| \geq \left\|g\right\|_1\). Finalmente consideremos o caso \(p = 1, q = \infty\). Seja \(g \in L_\infty\) e \(A_n \uparrow \Omega\) com \(0 < \mu(A_n) < \infty\). Consideremos \[\varepsilon(g)I_{[|g| \geq \left\|g\right\|_\infty-\eta]}I_{A_n}.\] \[\int_\Omega |\varepsilon(g)I_{[|g| \geq \left\|g\right\|_\infty-\eta]}I_{A_n}| d\mu =\] \[= \mu([|g| \geq \left\|g\right\|_\infty-\eta] \cap A_n).\] Escolhamos \(n\) suficientemente grande para que o membro direito seja positivo. Agora: \[\varphi(\varepsilon(g)I_{[|g| \geq \left\|g\right\|_\infty-\eta]}I_{A_n}) = \int_\Omega [|g| \geq \left\|g\right\|_\infty-\eta] \cap A_n |g| d\mu \geq\] \[\geq \mu\left(\left[|g| \geq \left\|g\right\|_\infty-\eta\right] \cap A_n\right) (\left\|g\right\|_\infty-\eta)\] \[\implies \| \varepsilon(g)I_{[|g| \geq \left\|g\right\|_\infty-\eta]}I_{A_n}\|_I (\left\|g\right\|_\infty-\eta).\]

Então \(\left\|\varphi\right\| \geq \left\|g\right\|_\infty - \eta \forall \eta > 0\), e portanto \(\left\|\varphi\right\| \geq \left\|g\right\|_\infty\). Seja agora \(\varphi\) uma função linear limitada sobre \(L_p\), com \(1 \leq p < \infty\). Suponhamos primeiro que \(\mu(\Omega) < \infty\). A fórmula \(\nu(A) = \varphi(I_A)\) define uma função de conjunto aditiva.

Como \[|\nu(A)| = |\varphi(I_A)| \leq \left\|\varphi\right\| \left\|I_A\right\|_p = \left\|\varphi\right\| (\mu(A))^{1/p},\] resulta que \(\nu\) é contínua superiormente no vazio, e portanto uma medida com sinal. Também a desigualdade anterior indica que \(\nu \ll \mu\). Pelo Teorema de Radon-Nikodym, existe uma função \(g\) tal que \[\forall A \in \mathscr{A} \nu(A) = \int_A g d\mu = \int_\Omega I_A g d\mu.\] Por linearidade resulta que se \(f\) é uma função simples \[\varphi(f) = \int_\Omega fg d\mu.\]

Vamos provar agora que \(g \in L_q\). Consideremos primeiramente o caso \(1 < p < \infty\). Seja \(0 \leq g_n \uparrow |g|\), onde \(g_n\) é simples e \(\in L_q\). \[\int_\Omega g_n^q d\mu \leq \int_\Omega |g| g_n^{q-1} d\mu =\] \[= \int_\Omega \varepsilon(g) g_n^{q-1} g d\mu =\] \[= \varphi(\varepsilon(g)g_n^{q-1}) \leq \left\|\varphi\right\| \left\|\varepsilon(g)g_n^{q-1}\right\|_p =\] \[= \left\|\varphi\right\| \left(\int_\Omega \left|(\varepsilon(g)g_n^{q-1})\right|^p\right)^{1/p} = \left\|\varphi\right\| \left(\int_\Omega g_n^q\right)^{1/p} =\] \[= \left\|\varphi\right\| \left\|g_n\right\|_q^{q/p} = \left\|\varphi\right\| \left\|g_n\right\|_q^{q-1}.\] Ou seja, \[\left\|g_n\right\|_q^q \leq \left\|\varphi\right\| \left\|g_n\right\|_q^{q-1}\] e portanto, \(\left\|g_n\right\|_q \leq \left\|\varphi\right\|\). Temos então que \(\left\|g\right\|_q \leq \left\|\varphi\right\|\) (Teorema da Convergência Monotônica). Consideremos agora o caso \(p=1 (q=\infty)\). Queremos provar que \(\left\|g\right\|_\infty \leq \left\|\varphi\right\|\).

Caso contrário, a função simples \(\varepsilon(g)I_{[|g| > \left\|\varphi\right\|]}\) seria diferente de 0, e \[\varphi(\varepsilon(g)I_{[|g| > \left\|\varphi\right\|]}) =\] \[= \int_{[|g| > \left\|\varphi\right\|]} |g| d\mu >\] \[> \left\|\varphi\right\| \mu([|g| > \left\|\varphi\right\|]) =\] \[= \left\|\varphi\right\| \left\|\varepsilon(g)I_{[|g| > \left\|\varphi\right\|]}\right\|_1\] o que seria uma contradição. Temos portanto, em geral, que \(g \in L_q\). Agora os funcionais lineares limitados \[\varphi \text{ e } f \mapsto \int_\Omega fg d\mu\] coincidem sobre toda função simples. Mas essas são densas em \(L_p\) e portanto, coincidem em todo \(L_p\), o que prova o resultado para \(\mu(\Omega) < \infty\). Consideremos, para o caso geral, uma sequência: \(A_n \uparrow \Omega\) com \(\mu(A_n) < \infty\). Pelo provado até agora para cada \(n\), existe uma função \(g \in L_q(L_q(A_n, \mathscr{A} \cap A_n, \mu|_{A_n}))\), tal que \[\forall f \in L_p, \varphi(I_{A_n}f) = \int_\Omega I_{A_n} g_n d\mu.\] Se \(m \leq n\), aplicando a identidade anterior a indicadores de subconjuntos de \(A_m\), vemos imediatamente que \(\mu([g_m \neq g_n] \cap A_m) = 0\). Podemos então supor que \(g_m = g_n\) sobre \(A_m\), e definir \(g\) sobre \(\Omega\) pela fórmula \[g(\omega) = g_n(\omega), \quad \text{se } \omega \in A_n.\] É fácil verificar agora que esta função \(g\) representa \(\varphi\).

Para o caso \(1 < p < \infty\), é possível suprimir a hipótese de que a medida seja \(\sigma\)-finita. Este fato é o conteúdo da seguinte:

PROPOSIÇÃO 6.4.1. Se \(1 < p < \infty\), a hipótese de que a medida \(\mu\) seja \(\sigma\)-finita pode ser suprimida.

Demonstração. Para a primeira parte do teorema, essa hipótese foi utilizada somente para provar que no caso \(p = 1, q = \infty\), \(\left\|\varphi\right\| = \left\|g\right\|_\infty\). Na segunda parte prova-se que, se \(\varphi\) é um funcional linear sobre \(L_p\), \(1 < p < \infty\), então existe um conjunto \(A\) com medida \(\sigma\)-finita, tal que se \(f \in L_p\) e \(f=0\) sobre \(A\), então \(\varphi(f) = 0\). Isso permite reduzir o problema geral ao caso anterior, considerando o espaço \[\left(A, \mathscr{A} \cap A, \mu|_{A \cap \mathscr{A}}\right),\] que é \(\sigma\)-finito. Seja \(f_n\) uma sequência de funções em \(L_p\), tais que \[\varphi(f_n) \geq \left\|\varphi\right\| \left(1-\frac{1}{n}\right),\] com \(\left\|f_n\right\| \leq 1\). Usando a desigualdade de Markov é fácil ver que existe \(A\), com medida \(\sigma\)-finita, tal que \(\forall n, f_n = 0\) sobre \(A^c\). Seja agora \(C\) um conjunto disjunto de \(A\) com medida finita. Temos então, para \(s \geq 0\), \[\left\|f_n \pm s I_C\right\|_p = \left(\int_\Omega |f_n \pm s I_C|^p\right)^{1/p} =\] \[= \left(\int_\Omega |f_n|^p + s^p \int_\Omega I_C^p\right)^{1/p} =\] \[= \left(\left\|f_n\right\|_p^p + s^p \mu(C)\right)^{1/p} \leq\] \[\leq (1 + s^p \mu(C))^{1/p}.\]

Agora, \[\varphi(\pm s I_C) = \varphi(\pm s I_C)-\varphi(f_n) + \varphi(f_n) =\] \[= \varphi(f_n \pm s I_C)-\varphi(f_n) \leq\] \[\leq \left|\varphi(f_n \pm s I_C)\right|-\left\|\varphi\right\| \left(1-\frac{1}{n}\right) \leq\] \[\leq \left\|\varphi\right\| \left\|f_n \pm s I_C\right\|_p - \left\|\varphi\right\| \left(1-\frac{1}{n}\right) =\] \[= \left\|\varphi\right\| \left\{(1 + s^p \mu(C))^{1/p} - \left(1-\frac{1}{n}\right)\right\}.\] Portanto, \[\left|\varphi(sI_C)\right| \leq \left\|\varphi\right\| \left\{(1 + s^p \mu(C))^{1/p} - \left(1-\frac{1}{n}\right)\right\}.\] Tomando o limite, quando \(n \to \infty\), \[\left|\varphi(sI_C)\right| \leq \left\|\varphi\right\| \left\{(1 + s^p \mu(C))^{1/p}-1\right\}.\] Dividindo por \(s\), temos \[\left|\varphi(I_C)\right| \leq \left\|\varphi\right\| \left\{\frac{(1 + s^p \mu(C))^{1/p}-1}{s}\right\}.\]

Usando a regra de L’Hospital é fácil ver que, quando \(s \to 0\), o membro direito converge a 0. Portanto, para todo \(C\) disjunto de \(A\) com medida finita, \(\varphi(I_C) = 0\). De aqui decorre o mesmo resultado para funções simples e por aproximação para funções \(f \in L_p\) com suporte disjunto de \(A\) como queríamos provar.

Com as mesmas técnicas usadas na demonstração do Teor. 6.4.1 podemos obter a proposição a seguir.

PROPOSIÇÃO 6.4.2. Seja \((\Omega, \mathscr{A}, \mu)\) um espaço de medida \(\sigma\)-finito. Para que uma funcional linear \(\varphi\) sobre \(L_\infty(\Omega, \mathscr{A}, \mu)\) seja da forma \[\varphi(f) = \int_\Omega fg d\mu,\] com \(g \in L_1(\Omega, \mathscr{A}, \mu)\), é necessário e suficiente que a função aditiva de conjunto \(A \mapsto \varphi(I_A)\) seja \(\sigma\)-aditiva sobre \(\mathscr{A}\).
