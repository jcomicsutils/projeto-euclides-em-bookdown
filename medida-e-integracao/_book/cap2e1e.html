Medida e Integração

2.1 Medidas

Seja \(\Omega\) um conjunto e \(\varnothing \neq \mathscr{A} \subseteq \mathbb{P}(\Omega)\).

DEFINIÇÃO 2.1.1. Uma função \(\mu: \mathscr{A} \to (-\infty, +\infty]\) é chamada uma função de conjunto.

Nota. Por motivos que veremos adiante, é necessário excluir um dos símbolos \(+\infty\) ou \(-\infty\). Decidimos excluir o último.

DEFINIÇÃO 2.1.2. Seja \(\mu\) uma função de conjunto. \(\mu\) é finitamente aditiva se, e somente se, \(A = \sum\limits_{i=1}^{n} A_i, n \in \mathbb{N}, A_i \in \mathscr{A}, A \in \mathscr{A} \Rightarrow \mu(A) = \sum\limits_{i=1}^{n} \mu(A_i)\). \(\mu\) é \(\sigma\)-aditiva se, e somente se, \(A = \sum\limits_{i=1}^{\infty} A_i, A_i \in \mathscr{A}, A \in \mathscr{A} \Rightarrow \mu(A) = \sum\limits_{i=1}^{\infty} \mu(A_i)\).

DEFINIÇÃO 2.1.3. Se \(\mathbb{S}\) é um semi-anel, \(\mu: \mathbb{S} \to [0, \infty]\) é uma medida finitamente aditiva se, e somente se, \(\mu(\varnothing) = 0\) e \(\mu\) é finitamente aditiva.

Uma medida finitamente aditiva que é \(\sigma\)-aditiva é chamada uma medida.

Observação 1. A definição de medida sobre semi-álgebra, anel, álgebra, \(\sigma\)-anel e \(\sigma\)-álgebra, é a mesma.

DEFINIÇÃO 2.1.4. Uma terna \((\Omega, \mathscr{A}, \mu)\), em que \(\Omega\) é um conjunto, \(\mathscr{A}\) é uma \(\sigma\)-álgebra contida em \(\mathbb{P}(\Omega)\) e \(\mu\) é uma medida sobre \(\mathscr{A}\), é chamada um espaço de medida. O par \((\Omega, \mathscr{A})\) é chamado um espaço mensurável.

(Nota. Em vários resultados é suficiente ter uma terna \((\Omega, \mathscr{A}, \mu)\) em que \(\mathscr{A}\) é só um \(\sigma\)-anel. Nesses casos o fato de \(\mathscr{A}\) ser \(\sigma\)-anel será mencionado explicitamente).

DEFINIÇÃO 2.1.5. Um espaço de medida \((\Omega, \mathscr{A}, \mu)\) tal que \(\mu(\Omega) = 1\), é denominado um espaço de probabilidades.

DEFINIÇÃO 2.1.6. Se \((\Omega, \mathscr{A}, \mu)\) é um espaço de medida e \(\forall A \in \mathscr{A}, A \subseteq \bigcup\limits_{i=1}^{\infty} A_i\) com \(\mu(A_i) < \infty, \forall i\), então \(\mu\) é chamada \(\sigma\)-finita. Se \(\mu(A) < \infty \forall A \in \mathscr{A}\), então \(\mu\) é chamada finita.

Em geral para uma classe arbitrária de conjuntos \(\mathscr{A}\) contidos em \(\Omega\) diremos que \(\mu: \mathscr{A} \to [-\infty, +\infty]\) é \(\sigma\)-finita (resp. finita) se \(\forall A \in \mathscr{A}, \exists\) uma seqüência de elementos de \(\mathscr{A}, \{A_i\}_{i=1,2,...}\) tal que \(A \subseteq \bigcup\limits_{i=1}^{\infty} A_i\) e \(|\mu(A_i)| < \infty \forall i\) (resp. \(|\mu(A)| < \infty, \forall A \in \mathscr{A}\)).

PROPOSIÇÃO 2.1.1. Seja \((\Omega, \mathscr{A}, \mu)\) um espaço de medida.

  1. \(A \subseteq B, \mu(B) < \infty \Rightarrow \mu(B-A) = \mu(B) - \mu(A).\)

  2. \(A_n \uparrow A \Rightarrow \mu(A) = \lim_n \mu(A_n).\)

  3. \(A_n \downarrow A\) e \(\exists n_0 \in \mathbb{N}\) tal que \(\mu(A_{n_0}) < \infty \Rightarrow \mu(A) = \lim_n \mu(A_n).\)

  4. \(\mu\left(\bigcup\limits_{n=1}^{\infty} A_n\right) \le \sum\limits_{n=1}^{\infty} \mu(A_n).\)

  5. Para toda sucessão \(\{A_n\}\) de elementos de \(\mathscr{A}\), \(\mu(\lim \inf A_n) \le \lim \inf \mu(A_n)\) e se, para algum \(n_0\), \(\mu\left(\bigcup\limits_{n=n_0}^{\infty} A_n\right) < \infty\), então \(\lim \sup \mu(A_n) \le \mu(\lim \sup A_n)\). Em particular se \((\Omega, \mathscr{A}, P)\) é um espaço de probabilidades, \(A_n \to A \Rightarrow P(A_n) \to P(A).\)

  6. \(\sum\limits_{n=1}^{\infty} \mu(A_n) < \infty \Rightarrow \mu(\lim \sup A_n) = 0.\)

(Nota. A Prop. 2.1.1 continuaria válida se \(\mathscr{A}\) fosse só um \(\sigma\)-anel).

Demonstração.

  1. \(B = A + (B-A) \Rightarrow \mu(B) = \mu(A) + \mu(B-A)\) e como \(\mu(A) \le \mu(B) < \infty\), podemos subtrair \(\mu(A)\) de ambos os membros, obtendo o resultado desejado.

  2. \(A = A_1 + \sum\limits_{i=2}^{\infty} (A_i - A_{i-1}) \Rightarrow\) \[ \begin{aligned} \Rightarrow \mu(A) &= \mu(A_1) + \sum_{i=2}^{\infty} \mu(A_i - A_{i-1}) = \\ &= \lim_{n \to \infty} \left[ \mu(A_1) + \sum_{i=2}^{n} \mu(A_i - A_{i-1}) \right] = \\ &= \lim_{n \to \infty} \mu \left[ A_1 + \sum_{i=2}^{n} (A_i - A_{i-1}) \right] = \\ &= \lim_{n \to \infty} \mu(A_n). \end{aligned} \]

  3. \[ A = A_{n_0} - \bigcup_{n=n_0}^{\infty} (A_{n_0} - A_n). \] Então, por 1, \(\mu(A) = \mu(A_{n_0}) - \mu\left[\bigcup_{n=n_0}^{\infty} (A_{n_0} - A_n)\right]\). Mas, como \(\{A_{n_0} - A_n\}_{n \ge n_0}\) é uma sucessão crescente, por 2, \[ \mu\left[\bigcup_{n=n_0}^{\infty} (A_{n_0} - A_n)\right] = \lim_n \mu(A_{n_0} - A_n) \] e então, \[ \mu(A) = \mu(A_{n_0}) - \lim_n[\mu(A_{n_0}) - \mu(A_n)] = \lim_n \mu(A_n). \]

  4. \[ \bigcup_{n=1}^{\infty} A_n = A_1 + \sum_{n=2}^{\infty} \left(A_n - \bigcup_{i=1}^{n-1} A_i\right) \] \[ \mu\left(\bigcup_{n=1}^{\infty} A_n\right) = \mu(A_1) + \sum_{n=2}^{\infty} \mu\left(A_n - \bigcup_{i=1}^{n-1} A_i\right) \le \] \[ \le \mu(A_1) + \sum_{n=2}^{\infty} \mu(A_n) = \sum_{n=1}^{\infty} \mu(A_n). \]

  5. \[ \lim \inf A_n = \bigcup_{n=1}^{\infty} \bigcap_{i=n}^{\infty} A_i. \] Como \(\left\{\bigcap\limits_{i=n}^{\infty} A_i\right\}_{n \in \mathbb{N}}\) é uma sucessão crescente, por 2, \(\mu(\lim \inf A_n) = \lim_n \mu\left(\bigcap\limits_{i=n}^{\infty} A_i\right) = \lim \inf_n \mu\left(\bigcap\limits_{i=n}^{\infty} A_i\right) \le \lim \inf \mu(A_n)\), pois \(\bigcap\limits_{i=n}^{\infty} A_i \subseteq A_n\). O caso de limite superior é análogo, e como obtemos uma seqüência decrescente, para usar o item 3 precisamos da hipótese de que para algum \(n_0\), \(\mu(\bigcup\limits_{n=n_0}^{\infty} A_n) < \infty\). Se \((\Omega, \mathscr{A}, P)\) é um espaço de probabilidade esta hipótese é desnecessária, porque a medida de qualquer conjunto é \(\le 1\). Nesse caso temos, \(P(\lim \inf A_n) \le \lim \inf P(A_n) \le \lim \sup P(A_n) \le P(\lim \sup A_n)\). Então, se \(A_n\) converge, \(\lim \inf A_n = \lim \sup A_n\) e todas as desigualdades se transformam em igualdades, o que implica que \(\lim P(A_n) = P(\lim A_n)\).

  6. \[ \lim \sup A_n = \bigcap_{n=1}^{\infty} \bigcup_{i=n}^{\infty} A_i. \] Como \(\{\bigcup_{i=n}^{\infty} A_i\}_{n \in \mathbb{N}}\) é uma sucessão decrescente e \[ \mu\left(\bigcup_{i=1}^{\infty} A_i\right) \le \sum_{i=1}^{\infty} \mu(A_i) < \infty, \] pelo item 3 temos: \[ \mu(\lim \sup A_n) = \lim_{n \to \infty} \mu\left(\bigcup_{i=n}^{\infty} A_i\right) \le \lim_{n \to \infty} \sum_{i=n}^{\infty} \mu(A_i) = 0, \] porque \(\sum_{i=1}^{\infty} \mu(A_n) < \infty\).
