Medida e Integração

Exercícios

  1. \((\Omega, \mathscr{A})\) espaço mensurável; \(\mu_1, \mu_2, \ldots, \mu_n\) medidas sobre \(\mathscr{A}\). Se \(a_1, a_2, \ldots, a_n\) são números reais não-negativos; então \(\mu = \sum\limits_{i=1}^{n} a_i\mu_i\) é uma medida.
  2. \((\Omega, \mathscr{A})\) espaço mensurável; \(P_1, P_2, \ldots\) uma seqüência de probabilidades sobre \(\mathscr{A}\). Se \(\alpha_1, \alpha_2, \ldots\) é uma seqüência de números não-negativos tais que \(\sum\limits_{i=1}^{\infty} \alpha_i = 1\), então \(\sum\limits_{i=1}^{\infty} \alpha_i P_i\) é uma probabilidade.
  3. Seja \(\Omega\) um conjunto enumerável e \(\mathscr{A} = \mathbb{P}(\Omega)\). Seja \(a_1, a_2, \ldots\) uma seqüência de números não-negativos tais que \(\sum\limits_{i=1}^{\infty} a_i < \infty\). Então a fórmula \(\mu(A) = \sum\limits_{i=1}^{\infty} a_i I_A(\omega_i)\) define uma medida finita sobre \(\mathscr{A}\). Reciprocamente toda medida finita é desta forma para alguma seqüência apropriada.
  4. Dê um exemplo mostrando que a condição de finitude na Prop. 2.1.1 parte 3) não pode ser eliminada.
  5. Seja \((\Omega, \mathscr{A}, \mu)\) um espaço de medida e sejam \[\eta_{\mu} = \{A: \mu(A) = 0\},\] \[\mathcal{N}_{\mu} = \{A: \mu(A) = 0 \quad \text{ou} \quad \mu(A^c) = 0\}.\] Prove que \(\eta_{\mu}\) é \(\sigma\)-anel e \(\mathcal{N}_{\mu}\) é \(\sigma\)-álgebra.
  6. Uma função \(\mu: \mathbb{S} \to [0, +\infty]\) é uma medida sobre o semi-anel \(\mathbb{S}\) se, e somente se,
  1. \(\mu(\varnothing) = 0\);
  2. \(A \in \mathbb{S}\), e \(A \subseteq \bigcup\limits_{n=1}^{\infty} A_n\) com \(A_n \in \mathbb{S} \implies \mu(A) \leq \sum\limits_{n=1}^{\infty} \mu(A_n)\);
  3. \(A \in \mathbb{S}\); \(A_1, A_2, \ldots, A_n \in \mathbb{S}\) e \(A = \sum\limits_{i=1}^{n} A_i \implies \mu(A) = \sum\limits_{i=1}^{n} \mu(A_i)\).
  1. Provar que a medida de Lebesgue do conjunto de Cantor é \(0\).
  2. Seja \((\Omega, \mathscr{A}, \mu)\) um espaço de medida. Seja \(\mathscr{A}_1 = \{A: A \in \mathscr{A} \text{ com } \mu(A) < \infty\}\). Definimos para \(A\) e \(B\) em \(\mathscr{A}_1\), \[d(A, B) = \mu(A \Delta B).\] Provar que:
  1. \((\mathscr{A}_1, d)\) é um espaço pseudométrico.
  2. Suponhamos que \(\mathscr{A}\) é gerada por um semi-anel \(\mathbb{S}\) finito ou enumerável, e que \(\mu\) é finita sobre \(\mathbb{S}\). Então \((\mathscr{A}_1, d)\) é um espaço topológico separável.
  1. Provar que se \((\Omega, \mathscr{A}, \mu)\) é um espaço de probabilidades, onde \(\Omega\) é um espaço polonês (um espaço métrico separável e completo), e \(\mathscr{A}\) é a \(\sigma\)-álgebra de Borel de \(\Omega\), então \(\forall \varepsilon > 0\), \(\exists K\) compacto, tal que \(\mu(K) > 1-\varepsilon\).
  2. Consideremos \((\mathbb{R}^d, \mathscr{A}, \lambda)\). Então, se \(A \in \mathscr{A}\) (mensurável Lebesgue), \[\lambda(A) = \inf\{\lambda(0): 0 \supseteq A, 0 \text{ aberto}\}\] \[= \sup\{\lambda(K): K \subseteq A, K \text{ compacto}\}.\]
  3. Seja \(\Omega = \{x: x \text{ racional}, 0 < x \leq 1\}\), \(\mathbb{S} = \{ (a, b]: (a, b] \subseteq \Omega, a \leq b \}\), \(\lambda((a, b]) = b-a\). Então \(\mathbb{S}\) é semi-álgebra, e \(\lambda\) sobre \(\mathbb{S}\) é uma função de conjunto, não-negativa e finitamente aditiva. Provar que \(\lambda\) não é \(\sigma\)-aditiva.
  4. Em \((\mathbb{R}^1, \lambda)\), construir \(\forall \varepsilon > 0\), um conjunto fechado tal que
  1. \(F = \varnothing\).
  2. \(\lambda(F^c) < \varepsilon\).
  1. Seja \(\mathscr{A}\) uma álgebra enumerável sobre \(\Omega\). Provar que \(\sigma(\mathscr{A})\) tem a potência do contínuo.
  2. Provar que a \(\sigma\)-álgebra de Lebesgue em \(\mathbb{R}^1\) tem a mesma cardinalidade (\(2^c\)) que a família de partes de \(\mathbb{R}^1\). Deduzir que existem conjuntos mensuráveis Lebesgue que não são Borelianos.
  3. (Critério de de la Vallée-Poussin). Provar que uma condição necessária e suficiente para que um conjunto limitado \(A\) seja mensurável Lebesgue e que \(\forall \varepsilon > 0\), \(\exists K\) compacto, \(K \subseteq A\) tal que \(\lambda^*(A-K) < \varepsilon\).
  4. Construir um conjunto mensurável \(A \subseteq [0, 1]\) tal que se \(U\) é um intervalo aberto contido em \([0, 1]\), \(\lambda(U \cap A) > 0\) e \(\lambda(U \cap A^c) > 0\).
  5. Seja \(\mu\) uma solução do problema fácil da medida. Provar:
  1. Se \(A \subseteq B \implies \mu(A) \leq \mu(B)\);
  2. \(\forall a \in \mathbb{R}^1 \mu(\{a\}) = 0\);
  3. \(\forall a, b \in \mathbb{R}^1\), \(a \leq b \mu([a, b]) = b-a\);
  4. Se \(\lambda\) é a medida de Lebesgue e \(0\) é um aberto, então \(\mu(0) \geq \lambda(0)\).

Seja \(\mu\), tal que se \(0 = \sum\limits_{k=1}^{\infty} (a_k, b_k)\), então \(\mu(0) = \sum\limits_{k=1}^{\infty} (b_k - a_k)\). Uma tal \(\mu\) é chamada regular.

  2. Defina uma função \(F: \mathbb{R}^1 \to \mathbb{R}^1\) tal que a medida de Lebesgue-Stieltjes gerada por \(F\) tenha as seguintes propriedades: \[\mu_F(\{0\}) = \frac{1}{2};\] \[\mu_F([0, \frac{1}{2}]) = 2;\] \[\mu_F((-\infty, -\frac{1}{2}]) = 0;\] \[\mu_F([1, x]) = 2x, \forall x > 1.\]
  3. Seja \(\Omega\) um conjunto. Para \(A \in \Omega\) indicaremos com \(\#(A)\) o número de elementos de \(A\) (\(+\infty\) se esse número não é finito). Provar que \((\Omega, \mathbb{P}(\Omega), \#)\) é um espaço de medida. A medida \(\#\) é chamada medida da contagem. Provar que \(\#\) é \(\sigma\)-finita se, e somente se, \(\Omega\) é enumerável.
  4. Considere \((\mathbb{N}, \mathbb{P}(\mathbb{N}), \#)\) (veja exerc. anterior). Definimos sobre \(\mathbb{P}(\mathbb{N})\), \[\mu(A) = \lim_{n \to \infty} \frac{1}{n} \# (A \cap \{1, 2, \ldots, n\})).\] Seja \(\mathscr{A} = \{A: A \subseteq \mathbb{N}, \lim_{n \to \infty} \frac{1}{n} \# (A \cap \{1, 2, \ldots, n\}) \text{ existe}\}\). Provar:
  1. \(\mathscr{A}\) não é sequer uma semi-álgebra;
  2. \(\mu\) é finitamente aditiva sobre \(\mathscr{A}\);
  3. Se \(A\) é finito \(\mu(A) = 0\);
  4. \(\mu\) não é \(\sigma\)-aditiva sobre \(\mathscr{A}\);
  5. \(\mu\) é uma medida exterior?

  1. Seja \((\Omega, \mathscr{A}, \mu)\) um espaço de medida com \(\mu\) \(\sigma\)-finita. Provar que o conjunto \(\{\omega: \mu(\{\omega\}) > 0\}\) é enumerável ao máximo (\(\{\omega\} \in \mathscr{A}, \forall \omega \in \Omega\)).

  2. Seja \((\Omega, \mathscr{A})\) um espaço mensurável e \(\{\mu_n\}_{n=1,2,\ldots}\) uma seqüência de medidas sobre \(\mathscr{A}\), tais que \(\mu_1 \leq \mu_2 \leq \mu_3 \leq \ldots\). Então \(\mu\) definida como o limite \(\mu(A) = \lim_n \mu_n(A)\) é uma medida.

  3. Considere o espaço \((\mathbb{N}, \mathbb{P}(\mathbb{N}), \#)\) (veja o Exerc. 20 do Cap. 2). Prove \(A_n \downarrow \varnothing \implies \#(A_n) \downarrow 0\).

  4. Seja \((\Omega, \mathscr{A}, \mu)\) um espaço de medida. Um conjunto \(A \in \mathscr{A}\) é chamado um átomo se \(\mu(A) > 0\) e para todo \(B, B \subseteq A\) resulta \(\mu(B) = 0\) ou \(\mu(B) = \mu(A)\), \(\mu\) é chamada não-atômica se não tem átomos. Prove que se \(\mu\) é não-atômica e \(A \in \mathscr{A}\), então \(\forall t, 0 \leq t \leq \mu(A)\), existe \(B \subseteq A\), tal que \(\mu(B) = t\).

  5. Sejam \(\mu\) e \(\nu\) duas medidas finitas sobre \((\mathbb{R}^1, \mathscr{B}^1)\). Se para todo \(x \in \mathbb{R}^1\), \(\mu((-\infty, x]) = \nu((-\infty, x])\), então \(\mu = \nu\) (\(\mathscr{B}^1 = \text{borelianos de } \mathbb{R}^1\)).

  6. Seja \(\Omega\) o conjunto dos racionais do intervalo \((0, 1]\). Seja \(\mathbb{S} = \{A: \text{existe } a \text{ e } b \in \Omega, a \leq b \text{ tal que } A = \{\omega: \omega \in \Omega, a < \omega \leq b\}\}\). \(\mathbb{S}\) é semi-álgebra. Seja \(\mathbb{R}\) a \(\sigma\)-álgebra gerada e \(\sigma(\mathbb{S})\) a \(\sigma\)-álgebra gerada, que neste caso coincide com \(\mathbb{P}(\Omega)\). Seja \(\mu_1 = \#\) (medida da contagem), e \(\mu_2 = 2 \#\). Então \(\mu_1 = \mu_2\) sobre \(\mathbb{R}\) mas \(\mu_1 \neq \mu_2\) (sobre \(\sigma(\mathbb{S}) = \mathbb{P}(\Omega)\)). Este exemplo prova que a condição de \(\sigma\)-finitude na Prop. 2.2.8 é essencial.

  7. Provar que um conjunto \(A\) tem medida de Lebesgue \(0\) se, e somente se, pode ser coberto com uma família enumerável de intervalos abertos de forma tal que: i) a soma total dos comprimentos dos intervalos seja finita; ii) cada ponto de \(A\) pertence a infinitos desses intervalos.

  8. Definição Dizemos que uma sequência de conjuntos \(\{A_n\}_{n=1,2,\ldots}\) tem a propriedade de interseção finita (p.i.f.) se \(\forall n, \bigcap\limits_{i=1}^{n} A_i \neq \varnothing\). Seja \(\mathscr{C}\) uma classe compacta e consideremos em \(\mathscr{C}\), uma sequência de conjuntos \(A_1, A_2, \ldots\) com a p.i.f.

    Para provar que \(\mathscr{C}_f\) é uma classe compacta devemos verificar que \(\bigcap\limits_{i=1}^{\infty} A_i \neq \varnothing\). Agora, \(\forall j, A_j = C_{j1} \cup C_{j2} \cup \ldots \cup C_{jk_j}, C_{ji} \in \mathscr{C}\). Como \(\mathscr{C}\) é classe compacta, basta obter uma sequência \(C_{1i_1}, C_{2i_2}, \ldots\) com a p.i.f., porque \(\bigcap\limits_{j=1}^{\infty} C_{ji_j} \subseteq \bigcap\limits_{j=1}^{\infty} A_j\).

    Essa sequência pode ser obtida definindo-se, indutivamente, índices \(i_1, i_2, \ldots\), de forma tal que \(\forall n \quad C_{1i_1}, C_{2i_2}, \ldots, C_{ni_n}, A_{n+1}, A_{n+2}, \ldots\) seja uma sequência com a p.i.f. Para construir essa sequência indutivamente, pode ser utilizado o seguinte:

    Lema Seja \(B_1, B_2, \ldots\) uma sequência com a p.i.f. Suponhamos que para um certo índice \(n_0\), \(B_{n_0} = C_1 \cup C_2 \cup \ldots \cup C_k\).

    Então existe um índice \(i, 1 \leq i \leq k\), tal que \(B_1, B_2, \ldots, B_{n_0-1}, C_i, B_{n_0+1}, \ldots\) tem a p.i.f.

    Demonstração. Caso contrário \(\forall i, \exists n_i > n_0\), tal que \(B_1 \cap B_2 \cap \ldots \cap B_{n_0-1} \cap C_i \cap \ldots \cap B_{n_i} = \varnothing\). Seja \(n = \max\limits_{1 \leq i \leq k} n_i\). Temos então \(\forall i, \quad i = 1, 2, \ldots, k, \quad B_1 \cap B_2 \cap \ldots \cap B_{n_0-1} \cap C_i \cap \ldots \cap B_n = \varnothing\).
