Medida e Integração

4.1 A integral. Teoremas básicos de convergência

Repetimos de forma ligeiramente modificada a Definição 3.5.

DEFINIÇÃO 4.1.1. Uma função mensurável é dita simples quando toma somente um número finito de valores finitos. Toda função simples pode ser representada na forma \[f = \sum\limits_{i=1}^n a_i I_{A_i}, \text{ onde os } A_i \in \mathscr{A}, \text{ são disjuntos e }\] \[\sum\limits_{i=1}^n A_i = \Omega; a_i \in \mathbb{R}.\] Esta representação não é única. (Se agregarmos a restrição \(a_i \neq a_j\), se \(i \neq j\), então seria única).

DEFINIÇÃO 4.1.2. Se \(f\) é uma função simples \(\geq 0\), \(f = \sum\limits_{i=1}^n a_i I_{A_i}\), define-se integral de \(f\) como \[\sum\limits_{i=1}^n a_i\mu(A_i)\] e usam-se indistintamente os símbolos \[I(f) = \int_\Omega f d\mu = \int_\Omega f(\omega) \mu(d\omega) = \int_\Omega f = \int f\] para indicar este número (lembre-se que \(+\infty \cdot 0 = 0\)).

Observação. \(I(f)\) está bem definida, no seguinte sentido: Se \(\sum\limits_{i=1}^n a_i I_{A_i} = \sum\limits_{j=1}^m b_j I_{B_j}\), então \(\sum\limits_{i=1}^n a_i\mu(A_i) = \sum\limits_{j=1}^m b_j\mu(B_j)\) (Veja o Exerc. 1).

PROPOSIÇÃO 4.1.1. Sejam \(f, g\) funções simples \(\geq 0\), e \(a \geq 0\) um número real,

  1. \(I(f) \geq 0\);

  2. \(f \leq g \implies I(f) \leq I(g)\);

  3. \(I(af) = aI(f)\);

  4. \(I(f+g) = I(f) + I(g)\).

Demonstração. Veja o Exerc. 2.

DEFINIÇÃO 4.1.3. Se \(A \in \mathscr{A}\) e \(f\) é uma função simples \(\geq 0\), por definição \[\int_A f = \int_\Omega f I_A.\]

PROPOSIÇÃO 4.1.2. A função \[A \mapsto \int_A f d\mu\] é uma medida sobre \(\mathscr{A}\).

Demonstração. Veja o Exerc. 3.

PROPOSIÇÃO 4.1.3. Se \(\{f_n\}\) é uma sucessão de funções simples \(\geq 0\), tal que \(f_n \downarrow 0\) e \(\mu\left(\left[f_{n_1} > 0\right]\right) < \infty\), para algum \(n_1 \in \mathbb{N}\), então \(I(f_n) \downarrow 0\).

Demonstração. Seja \(B = \left[f_{n_1} > 0\right]\). Se \(B = \varnothing\), o resultado é evidente. Suponhamos \(B \neq \varnothing\) e consideremos o espaço de medida \((B, \mathscr{A}, \mu|_{B})\) induzido por \((\Omega, \mathscr{A}, \mu)\) sobre \(B\). Como \(\{f_n\}\) é decrescente, para \(n \geq n_1\), temos \(\int_B f_n = \int_B f_n\). Por outro lado, como \(\{f_n\}_{n \geq n_1} \downarrow 0\) e \(\mu(B) < \infty\), pelo Teorema de Egoroff a convergência é quase uniforme. Seja \(\varepsilon > 0\). Então existe \(A \subseteq B\), tal que \[\mu(A) < \frac{\varepsilon}{2 \cdot \max f_{n_1}}\] e existe \(n_0 \geq n_1\), tal que \[n \geq n_0 \implies f_n(\omega) < \frac{\varepsilon}{2\mu(B)}\] para todo \(\omega \in B-A\). Escolhendo este \(n_0\), temos que \[n \geq n_0 \implies \int_B f_n = \int_A f_n + \int_{B-A} f_n \leq\] \[\leq (\max f_{n_1}) \cdot \mu(A) + \frac{\varepsilon}{2\mu(B)} \cdot \mu(B-A) < \varepsilon.\]

PROPOSIÇÃO 4.1.4. Se \(\{f_n\}\) é uma sucessão crescente de funções simples \(\geq 0\) e \(g\) é uma função simples \(\geq 0\), tal que \(\lim f_n \geq g\), então \(\lim \int f_n \geq \int g\).

Demonstração. Suponhamos que \(\mu\left(\left[g > 0\right]\right) < \infty\). Seja \(h_n = g-(g \wedge f_n)\), para todo \(n\). Então como \(h_n \downarrow 0\) e \(\mu\left(\left[h_1 > 0\right]\right) \leq \mu\left(\left[g > 0\right]\right) < \infty\), \(\int h_n \downarrow 0\). Então \[\int g = \lim \int (g \wedge f_n) \leq \lim \int f_n.\] Observe que esse último limite existe, porque a sucessão \(\{f_n\}\) é crescente. Supondo agora que \(\mu\left(\left[g > 0\right]\right) = \infty\), sejam \[m = \min\left\{g(\omega): \omega \in \left[g > 0\right]\right\} \text{ e } 0 < \varepsilon < m.\] Consideremos a sucessão \(\{A_n\}\), onde \[A_n = \left[\left. f_n > g-\varepsilon \right.\right] \cap \left[g > 0\right].\] Então \(A_n \uparrow \left[g > 0\right]\), como é fácil ver. Então \[\int f_n \geq \int_{A_n} f_n \geq (m-\varepsilon) \cdot \mu(A_n) \uparrow (m-\varepsilon)\mu\left(\left[g > 0\right]\right) = +\infty = \int g.\]

DEFINIÇÃO 4.1.4. Se \(f\) é uma função mensurável \(\geq 0\) e \(\{f_n\}\) é uma sucessão de funções simples \(\geq 0\), tal que \(f_n \uparrow f\), então, por definição, \(\int f = \lim \int f_n\).

Observação. \(\int f\) está bem definida no seguinte sentido: se \(\{g_n\}\) é uma sucessão de funções simples \(\geq 0\), tal que \(g_n \uparrow f\) então \[\lim \int f_n = \lim \int g_n.\] Isto é uma consequência da proposição anterior. Porque \(f = \lim f_n = \lim g_n \geq g_m \forall m\). Portanto \[\int f = \lim \int f_n \geq \int g_n \text{ (Prop. 4.1.4).}\] Como a desigualdade é válida \(\forall m\), temos que \(\lim \int f_n \geq \lim \int g_m\). A desigualdade contrária é provada de forma análoga.

PROPOSIÇÃO 4.1.5. Sejam \(f, g\) funções mensuráveis \(\geq 0\) e \(a \geq 0\). Então valem as mesmas afirmações da Prop. 4.1.1.

Demonstração. Veja o Exerc. 4.

PROPOSIÇÃO 4.1.6 (Teorema da Convergência Monótona). Seja \(\{f_n\}\) uma sucessão de funções, tais que \(f_n \uparrow f\), sendo as \(f_n\) e \(f\) mensuráveis \(\geq 0\). Então \[\int f_n \uparrow \int f.\]

Demonstração. Seja \(\{f_{n,m}\}\) uma sucessão de funções simples \(\geq 0\), tal que \(f_{n,m} \uparrow f_n\), para todo \(n\). Seja \[g_m = \sup_{1 \leq n \leq m} f_{n,m},\] para todo \(m\). Então as \(g_m\) são funções simples \(\geq 0\) e a sucessão \(\{g_m\}\) é crescente. Vamos mostrar que \(g_m \uparrow f\). Se \(1 \leq n \leq m\), \(f_{n,m} \leq g_m \leq f_m\). Tomando o limite, quando \(m \to \infty\), obtemos \[f_n \leq \lim_{m} g_m \leq f\] e fazendo agora \(n \to \infty\), \[f \leq \lim_{m} g_m \leq f.\] Então \(\int g_m \uparrow \int f\). Para \(1 \leq n \leq m\) \[\int f_{n,m} \leq \int g_m \leq \int f_m.\] Fazendo \(m \to \infty\), \[\int f_n \leq \int f \leq \lim_{m} \int f_m\] e, quando \(n \to \infty\), \[\lim_{n} \int f_n \leq \int f \leq \lim_{m} \int f_m.\] Então \(\int f_n \uparrow \int f\).

COROLÁRIO 4.1.1. Seja \(\{f_n\}\) uma sucessão de funções mensuráveis \(\geq 0\). Então \[\int \sum\limits_{n=1}^{\infty} f_n = \sum\limits_{n=1}^{\infty} \int f_n.\]

Demonstração. Segue da proposição anterior observando que \(\sum\limits_{n=1}^N f_n \uparrow \sum\limits_{n=1}^\infty f_n\).

Observação. Decorre do corolário acima, que se \(f\) é mensurável \(\geq 0\), a função \[A \in \mathscr{A} \mapsto \int_A f\] é \(\sigma\)-aditiva, porque \[\int_{\sum\limits_{n=1}^\infty A_n} f = \int_\Omega f \cdot I_{\sum\limits_{n=1}^\infty A_n} = \int_\Omega \sum\limits_{n=1}^\infty (f \cdot I_{A_n}) = \sum\limits_{n=1}^\infty \int_\Omega f \cdot I_{A_n} = \sum\limits_{n=1}^\infty \int_{A_n} f.\] Observação. Este argumento proporciona outra demonstração da Prop. 4.1.2.

DEFINIÇÃO 4.1.5. A função \(A \mapsto \int_A f d\mu\) é chamada integral indefinida da função \(f\). Denotaremos a função integral indefinida com os símbolos \(\int f\), \(\int f d\mu\), ou \(\mu(f)\).

COROLÁRIO 4.1.2. Se \(f\) é mensurável \(\geq 0\) e \(\int f < \infty\), \(\int f\) é absolutamente contínua em relação a \(\mu\), ou seja, para todo \(\varepsilon > 0\), existe \(\delta > 0\), tal que \(\mu(A) < \delta \implies \int_A f < \varepsilon\).

Demonstração. Seja \(g_n = f I_{\left[f < n\right]}\), para todo \(n\). Então \(0 \leq g_n \uparrow f\), implicando que \[\int f = \int g_n \uparrow \int f.\] Como \[\int f = \int_{\left[f < n\right]} f + \int_{\left[f \geq n\right]} f,\] \(\int_{\left[f \geq n\right]} f \downarrow 0\). Então dado \(\varepsilon > 0\), existe \(n_0\), tal que \(\int_{\left[f \geq n_0\right]} f < \varepsilon/2\). Ora, \[\int_A f = \int_{A \cap \left[f < n_0\right]} f + \int_{A \cap \left[f \geq n_0\right]} f \leq n_0\mu(A) + \frac{\varepsilon}{2}\] e então, tomando \(\delta = \frac{\varepsilon}{2n_0}\), \(\mu(A) < \delta \implies \int_A f < \varepsilon\).

DEFINIÇÃO 4.1.6. Seja \(f\) uma função mensurável. Dizemos que \(f\) é integrável (ou integrável Lebesgue) se, e somente se, \[\int f^+ < \infty \quad \text{e} \quad \int f^- < \infty;\] ou seja, \(\int |f| < \infty\). Dizemos que \(f\) é quase-integrável se, e somente se, \[\int f^+ < \infty \quad \text{ou} \quad \int f^- < \infty.\] Em qualquer dos casos acima, por definição, \[\int f = \int f^+ - \int f^-.\]

PROPOSIÇÃO 4.1.7. Sejam \(f\) e \(g\) funções quase integráveis, \(c \in \mathbb{R}^1\).

  1. \(\int f \in \mathbb{R}\); \(\int f \in \mathbb{R}^1\) se, e somente se, \(f\) é integrável;

  2. \(f \geq 0 \implies \int f \geq 0\);

  3. \(f \leq g \implies \int f \leq \int g\);

  4. \(\int cf = c \int f\);

  5. Se \(f+g\) está definida e \(f^+\) e \(g^+\) (ou \(f^-\) e \(g^-\) são integráveis, então \(f+g\) é quase integrável e \(\int f+g = \int f + \int g\);

  6. \(\left|\int f\right| \leq \int |f|\);

  7. Se \(\{f_n\}\) é uma sucessão de funções quase-integráveis, tal que \(f_n \uparrow f\) (respectivamente \(f_n \downarrow f\)) e \(f_{n_0}\) (respectivamente \(f_{n_0}\)) é integrável para algum \(n_0\), então \(\int f_n \uparrow \int f\) (respectivamente \(\int f_n \downarrow \int f\)).

Demonstração. Vamos provar somente e) e g). Para provar e) é suficiente mostrar que, se \(f_1, f_2\) são funções mensuráveis \(\geq 0\), sendo uma delas integrável, e \(f_1-f_2 \neq \infty - \infty\), então \(f_1-f_2\) é quase-integrável e \[\int \left(f_1-f_2\right) = \int f_1 - \int f_2\] porque \[f+g = \left(f^+-f^-\right) + \left(g^+-g^-\right) = \left(f^+ + g^+\right)-\left(f^- + g^-\right)\] e então \(f+g\) é quase-integrável e \[\int f+g = \int f^+ - \int f^- + \int g^+ - \int g^- = \int f + \int g.\] Provemos então a afirmação acima. Como \[0 \leq \left(f_1-f_2\right)^+ \leq f_1 \quad \text{e} \quad 0 \leq \left(f_1-f_2\right)^- \leq f_2,\] \(f_1-f_2\) é quase-integrável. \(f_1-f_2 = \left(f_1-f_2\right)^+-\left(f_1-f_2\right)^-\) \(\implies \left(f_1-f_2\right)^+ + f_2 = \left(f_1-f_2\right)^- + f_1 \implies\) \(\implies \int \left(f_1-f_2\right)^+ + \int f_2 = \int \left(f_1-f_2\right)^- + \int f_1 \implies\) \(\implies \int f_1 - \int f_2 = \int \left(f_1-f_2\right)^+ - \int \left(f_1-f_2\right)^- = \int \left(f_1-f_2\right)\).

Do item g) vamos provar apenas a 1ª afirmação, já que a prova da outra é análoga. Para todo \(n \geq n_0\), \(0 \leq f_n \leq f_{n_0}\) e isso implica que \(0 \leq \int f^- \leq \int f_{n_0}^-\). Então \(f\) e as \(f_n(n \geq n_0)\) são quase-integráveis, e \(f_n + f_{n_0}^- = f_n^- + f_{n_0}^- = f_n^+ + (f_{n_0}^- - f_n^-) \geq 0\) e \(f_n \uparrow f\), \(f_{n_0}^- \uparrow f_{n_0}^-\).

Então, pelo Teorema da Convergência Monótona, \[\int \left(f_n + f_{n_0}^-\right) \uparrow \int \left(f + f_{n_0}^-\right),\] ou seja, \[\int f_n + \int f_{n_0}^- \uparrow \int f + \int f_{n_0}^-,\] e então \[\int f_n \uparrow \int f.\]

PROPOSIÇÃO 4.1.8. Seja \(f\) mensurável \(\geq 0\). \(f = 0\) q. t. p. se, e somente se, \[\int f d\mu = 0.\]

Demonstração. Seja \(A_n = \left[f > \frac{1}{n}\right]\). \[\mu(A_n) = \int_{A_n} I_{A_n} d\mu = \int_{A_n} n\left(\frac{1}{n}I_{A_n}\right) d\mu \leq\] \[\leq n \int_{A_n} f d\mu = 0.\] Ou seja, para todo \(n \geq 1\), \(\mu(A_n) = 0\) e portanto \(\mu\left(\left[f > 0\right]\right) = 0\). Reciprocamente, se \(f = 0\) q.t.p., seja \(A = \left[f > 0\right]\); \(\mu(A) = 0\). Agora se \[g(\omega) = \begin{cases} +\infty & \omega \in A \\ 0 & \omega \in A^c, \end{cases}\] temos \(0 \leq f \leq g\) e portanto \(0 \leq \int f \leq \int g\). Como \(0 \leq n I_A \uparrow g\), temos \[0 = \int n I_A d\mu \uparrow \int g d\mu\] e portanto \(\int g d\mu = 0\), o que prova a proposição.

COROLÁRIO 4.1.3.

  1. Seja \(f \geq 0\) mensurável. \(A \in \mathscr{A}\), tal que \(\mu(A) = 0\). Então \(\int_A f d\mu = 0\).

Sejam \(f\) e \(g\) mensuráveis, e \(f = g\) q. t. p., então

  1. Se \(f \geq 0\), \(g \geq 0 \implies \int f d\mu = \int g d\mu\);

  2. \(\int f\) integrável \(\implies g\) integrável e \(\int f d\mu = \int g d\mu\).

Demonstração. Veja o Exerc. 5.

A seguinte proposição estende o Teorema da Convergência Monótona (e portanto a parte g) da Prop. 4.1.7), ao caso de convergência q. t. p.

PROPOSIÇÃO 4.1.9. \(0 \leq f_n \uparrow f\) em q. t. p., então \(0 \leq \int f_n \uparrow \int f\).

Demonstração. \(A = \left[f_n \not\to f\right]\); \(\mu(A) = 0\). Temos \[0 \leq I_{A^c} f_n \uparrow I_{A^c} f\] e utilizando o Teorema da Convergência Monótona \[0 \leq \int I_{A^c} f_n \uparrow \int I_{A^c} f = \int f.\] (A igualdade das integrais decorre do item b) do Cor. 4.1.3.

TEOREMA 4.1.1 (Fatou-Lebesgue). Seja \(\{f_n\}\) uma sucessão de funções mensuráveis e \(g\) e \(h\) são funções integráveis então:

  1. \(\int \liminf f_n \leq \liminf \int f_n\);

  2. \(\int \limsup f_n \geq \limsup \int f_n\);

  3. Se \(f_n \xrightarrow{\text{q.t.p.}} f\) e \(|f_n| \leq g\) q. t. p. para todo \(n\), então \(\int f = \lim \int f_n\).

Esse resultado é conhecido como Teorema da Convergência Dominada.

Demonstração. Vamos provar apenas o item (a) porque (b) é análoga e (c) decorre de (a) e (b). Temos que \(\left(\sup_{k \geq n} f_k\right)^+ \leq g^+\) em q. t. p e então, como \(g^+\) é integrável, \(\left(\sup_{k \geq n} f_k\right)^+\) é integrável para todo \(n\). Além disso, \(\left\{\sup_{k \geq n} f_k\right\}\) é uma sucessão decrescente que converge para \(\limsup f_n\). Então, pela versão com convergência em q. t. p. da Prop. 4.1.7, item g) (veja o comentário que precede a Prop. 4.1.9), temos \[\int \limsup f_n = \int \inf_{n \geq 1} \sup_{k \geq n} f_k = \lim_{n \to \infty} \int \sup_{k \geq n} f_k = \limsup \int f_n.\]

Observações.

  1. Nas condições da parte c) do teorema, podemos concluir também que \(\int |f_n-f| \to 0\), porque \(|f_n-f| \to 0\) q. t. p. e \(|f_n-f| \leq 2g\) q. t. p. Além disso vale também que \[\sup_{A \in \mathscr{A}} \left|\int_A f_n - \int_A f\right| \to 0,\] porque \[\left|\int_A f_n - \int_A f\right| \leq \int_A |f_n-f| \leq \int |f_n-f|.\]

  2. Em 1) a convergência q. t. p. pode ser substituída por convergência em medida. Senão vejamos:

Se \(\int |f_n-f| \not\to 0\), então existe \(\varepsilon > 0\) e uma sequência \(\{n_k\}\) de naturais, tal que \(\int |f_{n_k}-f| \geq \varepsilon\), para todo \(k\). Mas \(f_{n_k} \xrightarrow{\mu} f\), e portanto existe subsequência \(n_{k_j}\), tal que \(f_{n_{k_j}} \to f\) q. t. p., (Cor. 3.5 da Prop. 3.2.1) e, como \(|f_{n_{k_j}}| \leq g\), temos \(\int |f_{n_{k_j}}-f| \to 0\), o que é uma contradição.
